User Defined Data Arrays

 
Proposal for data structure:  UserDefinedDataArrays_t

UserDefinedDataArrays_t provides a location for arbitrary
user defined (numerical) data arrays of arbitrary size
to be written.  This node can appear under the following
nodes:

CGNSBase_t
Zone_t
GridCoordinates_t
FlowSolution_t
DiscreteData_t
ConvergenceHistory_t

It has the following properties:

Name: UserDefinedDataArrays# (or user defined)
Label: UserDefinedDataArrays_t
Data-Type: MT
Cardinality: 0,N
Child Nodes:
    Descriptor_t           (0,N)
    DataClass_t            (0,1)
    DimensionalUnits_t     (0,1)
    DataArray_t:
       Name: DataArray# (or user defined)
       Label: DataArray_t
       Data-Type: I4 or R4 or R8
       Dimensions: user defined
       Dimension Values: user defined
       Data: data quantity
       Cardinality: 0,N
